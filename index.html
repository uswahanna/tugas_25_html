Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artike-artikel yang ada di website kita, tag article bisa digunakan dengan element header dan footer.
    Karena tag article memiliki type block, artinya element ini memiliki lebar sesuai lebar browser jika ita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kia memiliki blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.
    Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
